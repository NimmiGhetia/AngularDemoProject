# AngularDemoProject

## Purpose

This is a demo project to demonstrate our Angular 8 knowledge.

## Concepts Covered

- Components
- Observables
- Services
- Routing
- Authentication
- REST Communication

## Application Details

This is a shopping cart application with the following components - 

- Sign In
- Sign Up
- Product Listing
- Product Details
- Shopping Cart

## Technology Stack Used

Front end - Angular 8
Back end - Firebase
